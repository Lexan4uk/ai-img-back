## Компоненты

### 1. Строка поиска

- Поиск по: имени, ID пользователя
- Фильтрация в реальном времени по подстроке

### 2. Таблица пользователей

| Колонка | Сортируемая | Описание |
|---------|:-----------:|----------|
| ★/☆ | — | Избранное |
| ID | ✓ | Идентификатор пользователя (сокращённый) |
| Имя | ✓ | Имя / псевдоним |
| Пол | ✓ | М / Ж / Другой |
| Возраст | ✓ | Число |
| Сессий | ✓ | Общее количество сессий |
| Устройств | ✓ | Количество уникальных экземпляров устройств |
| Посл. сессия | ✓ | Дата последней сессии (ДД.ММ.ГГГГ) |

**Сортировка по умолчанию:** По имени (A–Z)

**Пагинация:** 30 записей на страницу
