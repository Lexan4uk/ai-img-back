# S-07: Избранное

## Назначение

Хранение закреплённых элементов для быстрого доступа.
Данные об избранном хранятся исключительно в локальной SQLite — не синхронизируются с сервером.
Экран разделён на три вкладки по типу сущности.

## Компоненты

### 1. Вкладки (Tabs)

```
┌──────────────────┬────────────────────┬──────────────────┐
│  ★ Сессии (12)   │  ★ Устройства (3)  │ ★ Пользов. (5)  │
└──────────────────┴────────────────────┴──────────────────┘
```

- Число в скобках — текущее количество закреплённых элементов
- Активная вкладка выделяется цветом / подчёркиванием

### 2. Вкладка «Сессии»

Отображает закреплённые сессии (аналогично карточкам S-06):

| Элемент | Описание |
|---------|----------|
| ★ | Всегда заполненная (элемент в избранном) |
| Дата и время | Формат: ДД.ММ.ГГГГ ЧЧ:ММ |
| Пользователь | Имя |
| Устройство | Модель (сокращённый ID экземпляра) |
| Рука и палец | «Правая, Указательный» |
| Свайпы | Количество + консистентность |
| [Открепить] | Убрать из избранного |
| [Открыть →] | Перейти к S-08 |

### 3. Вкладка «Устройства»

Закреплённые экземпляры устройств:

| Элемент | Описание |
|---------|----------|
| ID экземпляра | Сокращённый UUID |
| Бренд + Модель | Полное название |
| Пользователь | Привязанный (если есть) |
| Кол-во сессий | Число |
| [Открепить] | Убрать из избранного |
| [Открыть →] | Перейти к S-08 с фильтром |

### 4. Вкладка «Пользователи»

Закреплённые пользователи:

| Элемент | Описание |
|---------|----------|
| ID | Идентификатор |
| Имя | Имя / псевдоним |
| Пол, Возраст | Демографические данные |
| Кол-во сессий | Число |
| [Открепить] | Убрать из избранного |
| [Открыть →] | Перейти к S-08 с фильтром |

## Взаимодействия

| # | Действие | Результат |
|---|----------|----------|
| 1 | Переключение вкладки | Отображение соответствующего списка |
| 2 | [Открепить] | Элемент удаляется из избранного, счётчик обновляется |
| 3 | [Открыть →] | Переход к соответствующему экрану |
| 4 | Клик по карточке | Аналогично [Открыть →] |
