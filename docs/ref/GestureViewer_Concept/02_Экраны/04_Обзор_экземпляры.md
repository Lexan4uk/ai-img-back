# S-04: Обзор по экземплярам устройств

## Назначение

Экран для выбора конкретного физического устройства (экземпляра).
Позволяет отследить все свайпы с одного конкретного телефона.

## Компоненты

### 1. Фильтры-dropdown

**Расположение:** Над таблицей

| Фильтр | Тип | Описание |
|--------|-----|----------|
| Бренд | Dropdown (одиночный выбор) | «Все» | Samsung | Xiaomi | … |
| Модель | Dropdown (зависимый) | Обновляется при выборе бренда |

### 2. Строка поиска

- Поиск по: ID экземпляра, модели, имени пользователя
- Фильтрация в реальном времени

### 3. Таблица экземпляров

| Колонка | Сортируемая | Описание |
|---------|:-----------:|----------|
| ★/☆ | — | Избранное |
| ID экземпляра | ✓ | Сокращённый UUID |
| Бренд | ✓ | Бренд производителя |
| Модель | ✓ | Название модели |
| Пользователь | ✓ | Привязанный пользователь (или «—» если нет) |
| Сессий | ✓ | Количество сессий |
| Последняя сессия | ✓ | Дата последней сессии |

**Сортировка:** Клик по заголовку колонки: ASC → DESC → без сортировки

**Пагинация:** 30 записей на страницу

## Взаимодействия

| # | Действие | Результат |
|---|----------|----------|
| 1 | Клик по строке экземпляра | Переход к S-08 с фильтром по экземпляру |

| 3 | Выбор бренда/модели в dropdown | Фильтрация таблицы |
| 4 | Сортировка по колонке | Перестроение таблицы |
| 5 | Ввод в поиск | Мгновенная фильтрация |
