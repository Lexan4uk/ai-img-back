# S-08: Просмотр сессии

## Назначение

Детальный просмотр содержимого одной сессии. Отображение всех свайпов
с их сырыми данными. Точка входа в визуализацию отдельных свайпов (S-09).

## Компоненты

### 1. Навигационная строка

```
◄ Назад к списку          Сессия #a3f2..7b1c          ★
```

| Элемент | Описание |
|---------|----------|
| ◄ Назад к списку | Возврат к предыдущему экрану (S-06, S-05, S-03 и т.д.) |
| ID сессии | Сокращённый UUID |
| ★ | Добавить/убрать из избранного |

### 2. Шапка сессии (информационная панель)

**Расположение:** Верхняя часть основной области, под навигацией

Сетка метаданных (2 колонки × 2 строки):

| Поле | Значение | Иконка |
|------|----------|--------|
| Пользователь | Имя (ID) | 👤 |
| Устройство | Модель | 📱 |
| Рука | Левая / Правая | ✋ |
| Палец | Название | 🖐 |
| Дата/время | ДД.ММ.ГГГГ ЧЧ:ММ | 📅 |
| Теги | Чипы | 🏷 |
| Свайпы | Количество | 📊 |
| Консистентность | Индикатор + процент | ✅/⚠️/🔴 |

**Элементы шапки кликабельны:**
- Клик по пользователю → переход к S-05 (фильтр по пользователю)
- Клик по устройству → переход к S-04 (фильтр по экземпляру)

### 3. Область свайпов (горизонтальная лента)

**Расположение:** Под шапкой, занимает оставшееся пространство

**Направление:** Горизонтальная прокрутка (слева направо = первый → последний)

#### Карточка свайпа

```
┌──────────────────────┐
│  #1                  │   ← Порядковый номер
│  P: 🟢  R: 🟢        │   ← Бейджи: давление + радиус
│  45 точек            │   ← Количество точек
│──────────────────────│
│ x   y   r1  r2  pr.  │   ← Заголовки (единая строка)
│ 120 340 5.2 3.1 .45  │
│ 122 342 5.0 3.0 .48  │   ← Сырые данные (горизонтальная прокрутка)
│ 125 345 5.5 3.2 .50  │
│ ...                  │
└──────────────────────┘
```

**Ширина карточки:** Фиксированная (~220px)

**Высота:** На всю доступную высоту (скроллится по вертикали внутри карточки при большом количестве точек)

**Бейджи оценки давления (P):**

| Значение | Бейдж | Цвет фона (акцент) |
|----------|-------|---------------------|
| `valid` | 🟢 Зачётный | Зелёный |
| `binary` | 🟡 Двоичный | Жёлтый |
| `invalid` | 🔴 Незачётный | Красный |

**Бейджи оценки радиусов (R):**

| Значение | Бейдж | Цвет фона (акцент) |
|----------|-------|---------------------|
| `valid` | 🟢 Зачётный | Зелёный |
| `circular` | 🔵 Круглый | Синий |
| `point` | 🟠 Точечный | Оранжевый |
| `invalid` | 🔴 Незачётный | Красный |

**Адаптация для малого количества точек:**

Если свайп содержит мало точек (≤ 5), данные отображаются в две колонки:

```
┌──────────────────────┐
│  #4                  │
│  P: 🔴  R: 🟠        │
│  1 точка             │
│──────────────────────│
│  x: 300    r1: 0.0   │
│  y: 250    r2: 0.0   │
│  p: 0.10   ts: 16... │
└──────────────────────┘
```

## Контекстное поведение

Экран S-08 открывается из разных мест с **предустановленными фильтрами**:

| Источник | Предустановка |
|----------|--------------|
| S-03 (По устройствам) | Все сессии модели X |
| S-04 (По экземплярам) | Все сессии экземпляра Y |
| S-05 (По пользователям) | Все сессии пользователя Z |
| S-06 (История) | Конкретная сессия |
| S-07 (Избранное) | Конкретная сессия |
| S-01 (Dashboard) | Конкретная сессия |
| S-02 (Фильтры) | Конкретная сессия |

Когда экран открыт из списка (S-03/S-04/S-05), отображается **список сессий** сверху, а при выборе сессии — её свайпы внизу. Когда открыта конкретная сессия — список скрыт.

## Взаимодействия

| # | Действие | Результат |
|---|----------|----------|
| 1 | Горизонтальная прокрутка | Навигация по свайпам сессии |
| 2 | Клик по карточке свайпа | Открытие модального окна S-09 |
| 3 | ★ в навигации | Добавить/убрать сессию из избранного |
| 4 | ◄ Назад | Возврат к предыдущему экрану |
| 5 | Клик по имени пользователя | Переход к S-05 |
| 6 | Клик по устройству | Переход к S-04 |

## Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `←` / `→` | Прокрутка к предыдущему/следующему свайпу |
| `Enter` | Открыть визуализацию выделенного свайпа |
| `Backspace` | Назад к списку |
| `F` | Добавить/убрать из избранного |
