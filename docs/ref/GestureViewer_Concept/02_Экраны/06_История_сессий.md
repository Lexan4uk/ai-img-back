## Компоненты

### 1. Быстрый фильтр по дате

**Расположение:** Верхняя часть основной области

```
[Сегодня]  [Неделя]  [Месяц]  [Произвольный ▼]
```

| Кнопка | Фильтр |
|--------|--------|
| Сегодня | `created_at >= начало текущего дня` |
| Неделя | `created_at >= 7 дней назад` |
| Месяц | `created_at >= 30 дней назад` |
| Произвольный | Date range picker (календарь от–до) |

Активная кнопка выделяется цветом. Без выбора — показываются все сессии.

### 2. Карточка сессии

Каждая сессия отображается как карточка с полной метаинформацией:

| Элемент | Описание |
|---------|----------|
| ★/☆ | Избранное (слева) |
| Дата и время | Формат: ДД.ММ.ГГГГ ЧЧ:ММ |
| Пользователь | Иконка 👤 + имя |
| Устройство | Иконка 📱 + модель + (сокращённый ID экземпляра) |
| Рука и палец | Иконка ✋ + «Правая, Указательный» |
| Теги | Чипы (chips): [тест] [лаборатория] |
| Кол-во свайпов | Иконка 📊 + число |
| Консистентность | ✅ / ⚠️ кол-во (%) / 🔴 кол-во (%) |

### 3. Пагинация

- 30 карточек на страницу
- Альтернатива: infinite scroll (загрузка при прокрутке)
- Счётчик общего количества: `Всего: 12 450`
