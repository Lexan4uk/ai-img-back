# 10. Глобальная навигация и общие UI-паттерны

## Назначение

Описание элементов интерфейса, общих для всех экранов: боковая навигация,
верхняя панель, статусная строка и переиспользуемые компоненты.

---

## Общая структура окна

```
┌────────────────────────────────────────────────────────────────────────────────┐
│                          HEADER (верхняя панель)                             │
│  [Лого / Название]          [Хлебные крошки]           [🔍 Быстрый поиск]   │
├──────────────┬─────────────────────────────────────────────────────────────────┤
│              │                                                               │
│   SIDEBAR    │                 ОСНОВНАЯ ОБЛАСТЬ                              │
│  (навигация) │                   (content)                                   │
│              │                                                               │
│  Фиксир.     │  Скроллируемая                                                │
│  ~200px      │  Адаптивная по ширине                                         │
│              │                                                               │
│              │                                                               │
│              │                                                               │
│              │                                                               │
├──────────────┴─────────────────────────────────────────────────────────────────┤
│                           FOOTER (статусная строка)                          │
└────────────────────────────────────────────────────────────────────────────────┘
```

---

## 1. Header (верхняя панель)

**Высота:** 48–56px, фиксированная

| Элемент | Расположение | Описание |
|---------|-------------|----------|
| Логотип / Название | Слева | «GestureViewer» — брендинг приложения |
| Хлебные крошки | Центр | Навигационная цепочка: `Dashboard > По устройствам > Galaxy S23` |
| Быстрый поиск | Справа | Строка поиска с автодополнением (см. S-01) |
| Тема | Справа | Переключатель: 🌙 / ☀️ (тёмная/светлая) |

### Хлебные крошки (Breadcrumbs)

```
Dashboard  >  По устройствам  >  Samsung Galaxy S23  >  Сессия #a3f2..
```

- Каждый элемент кликабельный → переход к соответствующему экрану
- Последний элемент (текущий) — не кликабельный, выделен жирным

---

## 2. Sidebar (боковая навигация)

**Ширина:** ~200px (развёрнутая), ~56px (свёрнутая)

```
Развёрнутая:                    Свёрнутая:
┌──────────────────┐           ┌────────┐
│  📊 Dashboard    │           │  📊    │
│  📱 Устройства   │           │  📱    │
│  📋 Экземпляры   │           │  📋    │
│  👤 Пользователи │           │  👤    │
│  📅 История      │           │  📅    │
│  ★  Избранное    │           │  ★     │
│  🔍 Фильтры     │           │  🔍    │
│                  │           │        │
│                  │           │        │
│  [≡] Свернуть    │           │  [≡]   │
└──────────────────┘           └────────┘
```

**Поведение:**
- Активный пункт: фоновая подсветка + акцентная полоска слева (3px)
- Hover: подсветка фона (lighter)
- Кнопка [≡] — toggle свёрнутая/развёрнутая
- В свёрнутом виде: только иконки, tooltip при наведении
- Состояние (свёрнутая/развёрнутая) сохраняется в localStorage

---

## 3. Footer (статусная строка)

**Высота:** 24–28px, фиксированная

```
┌────────────────────────────────────────────────────────────────────────────────┐
│  SQLite ● Подключено  │  Записей: 12 450  │  Посл. обновление: 12.02.2026    │
└────────────────────────────────────────────────────────────────────────────────┘
```

| Элемент | Описание |
|---------|----------|
| Тип БД | `SQLite` (в будущем: `PostgreSQL`) |
| Статус | ● Подключено (зелёный) / ● Ошибка (красный) |
| Кол-во записей | Общее число сессий в БД |
| Посл. обновление | Дата/время последней синхронизации или загрузки |

## 4. Переиспользуемые компоненты

### Таблица с сортировкой

Используется на: S-02, S-04, S-05

```
┌───────────────────────────────────────────────────┐
│  Колонка A  ▼ │  Колонка B  │  Колонка C  ▲ │    │
│───────────────┼─────────────┼────────────────┼────│
│  значение     │  значение   │  значение      │    │
│  значение     │  значение   │  значение      │    │
└───────────────────────────────────────────────────┘
▲ = ASC, ▼ = DESC, без стрелки = без сортировки
```

### Пагинация

```
◄  1  2  3  ...  42  ►     Показано: 30 из 1 247
```

### Звёздочка избранного

```
☆ (не в избранном) ←→ ★ (в избранном)
```

### Чипы тегов

```
[тег1 ×]  [тег2 ×]  [тег3 ×]
```

### Индикатор консистентности

```
✅ Консистентно (100%)
⚠️ 2 проблемных (17%)
🔴 18 некорректных (75%)
```
