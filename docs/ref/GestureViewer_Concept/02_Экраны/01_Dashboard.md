# S-01: Главный экран / Dashboard

## Назначение

Точка входа в приложение. Обеспечивает:
- Общую сводку по базе данных
- Быстрый доступ ко всем разделам через навигацию
- Быстрый поиск и переход к последним сессиям

## Компоненты

### 1. Боковая панель навигации (Sidebar)

**Расположение:** Левая сторона, фиксированная ширина ~200px

| Пункт | Иконка | Переход |
|-------|--------|---------|
| Dashboard | 📊 | S-01 (текущий) |
| По устройствам | 📱 | S-03 |
| По экземплярам | 📋 | S-04 |
| По пользователям | 👤 | S-05 |
| История | 📅 | S-06 |
| Избранное | ★ | S-07 |
| Фильтры | 🔍 | S-02 |

**Поведение:**
- Активный пункт выделяется фоном и акцентной полосой слева
- Sidebar присутствует на ВСЕХ экранах (глобальная навигация)
- В свёрнутом состоянии показывает только иконки

### 2. Строка быстрого поиска

**Расположение:** Верхняя панель (Header), справа

**Функционал:**
- Поиск по: ID сессии, имени пользователя, модели устройства
- Автодополнение: выпадающий список совпадений при вводе от 2 символов
- Результаты группируются по типу: `[Сессия]`, `[Пользователь]`, `[Устройство]`
- Клик по результату → переход к соответствующему экрану

```

```

### 3. Карточки сводной статистики

**Расположение:** Основная область, верхняя часть (сетка 2×2 или 4×1)

| Карточка | Значение | Клик → |
|----------|----------|--------|
| Общее кол-во сессий | `COUNT(sessions)` | S-06 (История) |
| Уникальные пользователи | `COUNT(DISTINCT users)` | S-05 |
| Модели устройств | `COUNT(DISTINCT device_models)` | S-03 |
| Экземпляры устройств | `COUNT(DISTINCT device_instances)` | S-04 |

**Визуальное оформление:**
- Крупное число (24–32pt, accent color)
- Подпись под числом (12–14pt, secondary text)
- Hover-эффект: подсветка + cursor pointer

### 4. Блок последних сессий

**Расположение:** Основная область, под карточками статистики

**Содержимое:** 5–10 последних сессий (по `created_at` DESC)

| Колонка | Описание |
|---------|----------|
| ID | Сокращённый UUID (первые 6 символов) |
| Пользователь | Имя |
| Устройство | Модель |
| Рука | left/right |
| Палец | Название пальца |
| Консистентность | ✅ / ⚠️ / 🔴 |

**Действие:** Клик по строке → переход к S-08 (Просмотр сессии)

### 5. Статусная строка (Footer)

**Расположение:** Нижняя часть окна

**Показывает:**
- Тип подключения: `SQLite ● Подключено`
- Общее количество записей
- Дата/время последнего обновления данных

## Взаимодействия

| # | Действие | Результат |
|---|----------|----------|
| 1 | Клик по пункту навигации | Переход к соответствующему экрану |
| 2 | Клик по карточке статистики | Переход к списковому экрану |
| 3 | Клик по сессии в «Последних» | Переход к S-08 |
| 4 | Ввод в поисковую строку | Автодополнение + фильтрация |
| 5 | Клик по результату поиска | Переход к найденной сущности |

## Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl+K` / `Ctrl+F` | Фокус на строку поиска |
| `1`–`7` | Переход по пунктам навигации (в sidebar) |
| `Esc` | Закрыть dropdown поиска |
