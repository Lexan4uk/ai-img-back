## Компоненты

### 1. Вкладки фильтрации по бренду

**Расположение:** Верхняя часть основной области

```
┌────────┬──────────┬───────────┬────────────┬──────────┐
│  Все   │ Samsung  │  Xiaomi   │  Realme    │  Google  │ ...
└────────┴──────────┴───────────┴────────────┴──────────┘
```

- «Все» — показать модели всех брендов, сгруппированные по секциям
- Клик по бренду — показать только модели этого бренда
- Количество вкладок формируется динамически из БД
- При большом количестве брендов — горизонтальная прокрутка вкладок

### 2. Строка поиска по модели

**Расположение:** Справа от заголовка

- Фильтрация в реальном времени по названию модели
- Работает внутри выбранного бренда (или по всем, если «Все»)

### 3. Сортировка

| Критерий | Описание |
|----------|----------|
| По имени (A–Z) | Алфавитная сортировка названий моделей |
| По кол-ву сессий | Больше сессий — выше |
| По дате последней сессии | Недавно активные — выше |

### 4. Список моделей

**Группировка:** По секциям-брендам (если «Все»), без группировки при фильтре по бренду

**Колонки строки модели:**

| Колонка | Описание |
|---------|----------|
| ★/☆ | Добавить в избранное (экземпляры, не модель) |
| Название модели | Полное название |
| Кол-во сессий | Общее число сессий на всех экземплярах |
| Кол-во экземпляров | Уникальные физические устройства |
| Дата последней сессии | Формат: ДД.ММ.ГГГГ |

## Взаимодействия

| # | Действие | Результат |
|---|----------|----------|
| 1 | Клик по вкладке бренда | Фильтрация моделей по бренду |
| 2 | Клик по строке модели | Переход к S-06/S-08 с фильтром по модели |
| 3 | Клик по ★ | Добавить/убрать из избранного (S-07) |
| 4 | Ввод в поиск | Мгновенная фильтрация по подстроке |
| 5 | Смена сортировки | Перестроение списка |
