# S-02: Расширенные фильтры

## Назначение

Экран для комплексной фильтрации сессий по произвольной комбинации параметров.
Все фильтры объединяются по AND-логике. Результаты обновляются в реальном времени.

## Компоненты

### 1. Чип-панель активных фильтров

**Расположение:** Верхняя часть основной области, под header

```
[Samsung ×]  [Правая рука ×]  [Указательный ×]  [Возраст: 18–35 ×]    [Сбросить все]
```

- Каждый активный фильтр отображается как чип (chip) с кнопкой удаления `×`
- Клик по `×` — удаление конкретного фильтра, обновление результатов
- «Сбросить все» — очистка всех фильтров

### 2. Панель фильтров (левая часть)

**Ширина:** ~280px, скроллируемая

Фильтры организованы в раскрываемые группы (accordion):

#### Группа: Устройство

| Фильтр | Тип компонента | Описание |
|--------|---------------|----------|
| Бренд | Чекбоксы (мультивыбор) | Samsung, Xiaomi, Realme, Pixel… |
| Модель | Чекбоксы (мультивыбор) | Зависит от выбранного бренда (каскадная фильтрация) |
| Экземпляр | Поле ввода с автодополне­нием | Поиск по ID экземпляра |

**Каскадная зависимость:**
```
Бренд выбран: [✓] Samsung
  └─► Модели обновляются: Galaxy S23, Galaxy A54, Galaxy S24 Ultra…

Бренд не выбран:
  └─► Показаны все модели всех брендов
```

#### Группа: Пользователь

| Фильтр | Тип компонента | Описание |
|--------|---------------|----------|
| Имя / ID | Текстовое поле с автодополнением | Поиск по подстроке |
| Пол | Чекбоксы | Мужской, Женский, Другой |
| Возраст | Двойной слайдер (range slider) | От–До (мин. 0, макс. 100) |

#### Группа: Параметры сессии

| Фильтр | Тип компонента | Описание |
|--------|---------------|----------|
| Рука | Чекбоксы | Левая, Правая |
| Палец | Чекбоксы | Большой, Указательный, Средний, Безымянный, Мизинец |
| Теги | Облако тегов (cloud) с мультивыбором | Все уникальные теги из БД |
| Дата | Date range picker | Диапазон дат проведения сессии |

#### Группа: Оценки качества свайпов

| Фильтр | Тип компонента | Описание |
|--------|---------------|----------|
| Оценка давления | Чекбоксы (мультивыбор) | Зачётный (`valid`), Двоичный (`binary`), Незачётный (`invalid`) |
| Оценка радиусов | Чекбоксы (мультивыбор) | Зачётный (`valid`), Круглый (`circular`), Точечный (`point`), Незачётный (`invalid`) |
| Консистентность | Чекбокс | Только полностью консистентные сессии (100%) |

### 3. Область результатов (правая часть)

**Таблица сессий** (аналогична S-06, но управляется фильтрами):

| Колонка | Сортируемая | Описание |
|---------|:-----------:|----------|
| ID | ✓ | Сокращённый UUID |
| Пользователь | ✓ | Имя |
| Устройство | ✓ | Модель |
| Рука | ✓ | Левая / Правая |
| Палец | ✓ | Название |
| Свайпов | ✓ | Количество |
| Дата | ✓ | Дата/время |
| Качество | ✓ | ✅ / ⚠️ / 🔴 |

**Счётчик:** `Найдено: 1 247` — обновляется при каждом изменении фильтров (debounce 300ms)

**Пагинация:** Постраничная навигация снизу таблицы (30 записей на страницу)

## Взаимодействия

| # | Действие | Результат |
|---|----------|----------|
| 1 | Выбор значения фильтра | Таблица обновляется (debounce 300мс), чип добавляется |
| 2 | Удаление чипа (×) | Фильтр снимается, таблица обновляется |
| 3 | «Сбросить все» | Все фильтры очищены, полный список |
| 4 | Клик по сессии в таблице | Переход к S-08 |
| 5 | «Сохранить набор фильтров» | Сохранение текущей комбинации с именем (локально) |
| 6 | Сортировка по колонке | Клик по заголовку: ASC → DESC → без сортировки |

## Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl+Shift+F` | Фокус на первый фильтр |
| `Esc` | Сброс всех фильтров |
| `Enter` | Перейти к первой сессии в результатах |
