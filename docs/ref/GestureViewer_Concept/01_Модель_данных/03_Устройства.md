# 03. Сущности: Модель устройства и Экземпляр устройства

## Назначение

Два уровня абстракции устройств:
- **DeviceModel** — модель телефона (Samsung Galaxy S23, Xiaomi Redmi Note 12 и т.д.)
- **DeviceInstance** — конкретный физический экземпляр устройства

Разделение позволяет:
- Группировать данные по моделям (сравнение поведения на разных моделях)
- Отслеживать конкретный телефон (одно устройство, несколько пользователей)

---

## DeviceModel — Модель устройства

### Поля

| Поле | Тип | Обязательное | Описание |
|------|-----|:---:|----------|
| `id` | TEXT (UUID) | ✓ | Уникальный идентификатор модели |
| `brand` | TEXT | ✓ | Бренд производителя (Samsung, Xiaomi, Realme…) |
| `model_name` | TEXT | ✓ | Название модели (Galaxy S23, Redmi Note 12…) |

### Бизнес-правила

- Бренд используется для группировки на экране S-03
- Связка `brand + model_name` уникальна

---

## DeviceInstance — Экземпляр устройства

### Поля

| Поле | Тип | Обязательное | Описание |
|------|-----|:---:|----------|
| `id` | TEXT (UUID) | ✓ | Уникальный идентификатор экземпляра |
| `device_model_id` | TEXT (FK) | ✓ | Ссылка на DeviceModel |
| `created_at` | TEXT (ISO-8601) | ✓ | Дата первого появления в системе |

### Бизнес-правила

- Экземпляр привязан ровно к одной модели
- У экземпляра может быть несколько пользователей (телефон может быть передан)
- ID экземпляра приходит с устройства (ANDROID_ID или аналог)

---

## Диаграмма связей

```
DeviceModel ──┐
              │ 1 : N
              ▼
       DeviceInstance ──┐
                       │ 1 : N
                       ▼
                    Session
```

## Отображение в UI

| Экран | DeviceModel | DeviceInstance |
|-------|-------------|---------------|
| S-03 (По устройствам) | Список моделей, сгруппирован по бренду. Показывает: название, кол-во сессий, кол-во экземпляров, дата послед. сессии | — |
| S-04 (По экземплярам) | Колонки «бренд» и «модель» | Таблица экземпляров: ID, привяз. пользователь, кол-во сессий, дата послед. сессии |
| S-06 (История) | Модель в карточке сессии | ID экземпляра в карточке |
| S-08 (Просмотр сессии) | Модель в шапке | ID экземпляра в шапке |
| S-02 (Фильтры) | Мультивыбор бренда → модели | Мультивыбор по ID |
| S-07 (Избранное) | Вкладка «Избранные устройства» | Вкладка «Избранные устройства» |

## Агрегаты для UI

| Агрегат | Для кого | Описание |
|---------|----------|----------|
| `session_count` | DeviceModel | Общее число сессий на всех экземплярах модели |
| `instance_count` | DeviceModel | Число уникальных экземпляров |
| `last_session_date` | DeviceModel / Instance | Дата последней сессии |
| `linked_user_name` | DeviceInstance | Имя пользователя (если есть привязка) |
