# 06. Сущность: Избранное (Favorite)

## Назначение

Полиморфная сущность для хранения пользовательских закладок.
Позволяет закрепить для быстрого доступа: сессии, экземпляры устройств, пользователей.
Данные хранятся исключительно локально в SQLite.

## Поля

| Поле | Тип | Обязательное | Описание |
|------|-----|:---:|----------|
| `id` | INTEGER (PK, autoincrement) | ✓ | Локальный идентификатор |
| `entity_type` | TEXT | ✓ | Тип сущности: `"session"`, `"device_instance"`, `"user"` |
| `entity_id` | TEXT | ✓ | ID целевой сущности |
| `created_at` | TEXT (ISO-8601) | ✓ | Дата добавления в избранное |

## Бизнес-правила

- Пара `(entity_type, entity_id)` уникальна — нельзя добавить одну сущность дважды
- Добавление/удаление из избранного — мгновенное, без подтверждения
- Избранное НЕ синхронизируется с сервером — это локальные данные клиента
- При удалении сущности из основной БД (например, при переиндексации) — запись в Favorite становится «висячей». UI должен это обрабатывать корректно (скрывать или помечать как недоступную)

## Полиморфная связь

```
Favorite.entity_type = "session"         → Session.id
Favorite.entity_type = "device_instance" → DeviceInstance.id
Favorite.entity_type = "user"            → User.id
```

## Отображение в UI

### Экран S-07 (Избранное) — три вкладки:

```
┌───────────────────┬────────────────────┬────────────────────┐
│  ★ Сессии (12)    │  ★ Устройства (3)  │  ★ Пользователи(5) │
├───────────────────┴────────────────────┴────────────────────┤
│                                                            │
│  Список закреплённых элементов текущей вкладки             │
│  [★ Открепить]  [→ Перейти]                                │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

### Звёздочка (★) на других экранах:

| Экран | Элемент | Действие по клику |
|-------|---------|------------------|
| S-03 (По устройствам) | ★ рядом с моделью | *Примечание: модель не добавляется, только экземпляры* |
| S-04 (По экземплярам) | ★ рядом с экземпляром | Добавить/убрать DeviceInstance |
| S-05 (По пользователям) | ★ рядом с пользователем | Добавить/убрать User |
| S-06 (История) | ★ рядом с сессией | Добавить/убрать Session |
| S-08 (Просмотр сессии) | ★ в шапке | Добавить/убрать Session |

## Индикация состояния

- ★ (заполненная) — элемент в избранном
- ☆ (пустая) — элемент не в избранном
- Toggle мгновенный, без модального подтверждения
