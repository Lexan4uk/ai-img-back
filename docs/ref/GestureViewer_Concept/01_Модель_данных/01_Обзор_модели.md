# 01. Обзор модели данных

## Назначение

Описание всех сущностей, их полей и взаимосвязей в системе визуализации свайпов.
Данные поступают с Android-устройств, где установлено приложение-сборщик.
Desktop-клиент хранит локальную копию данных и работает автономно.

## Направление связей

| Связь | Тип | Описание |
|-------|-----|----------|
| User → Session | 1 : N | У одного пользователя может быть много сессий |
| DeviceModel → DeviceInstance | 1 : N | Модель устройства объединяет конкретные экземпляры |
| DeviceInstance → Session | 1 : N | На одном экземпляре проведено много сессий |
| Session → Swipe | 1 : N | Сессия содержит упорядоченный набор свайпов |
| Swipe → SwipePoint | 1 : N | Свайп состоит из точек (трека) |
| Favorite → * | Полиморф. | Избранное ссылается на любую из 3 сущностей |
