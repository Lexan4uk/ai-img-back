# 03. Навигация и общие паттерны

## Общая структура окна

```
┌─────────┬──────────────────────────────────────────────────────────────┐
│         │  Header (заголовок, тема, настройки)                 40px   │
│ SIDEBAR │──────────────────────────────────────────────────────────────│
│         │                                                              │
│  [≡]    │                                                              │
│         │                     Content Area                             │
│  🖼     │                (экран — S-01 или S-02)                       │
│ Галерея │                                                              │
│         │                                                              │
│  ⚡     │                                                              │
│Генера-  │                                                              │
│  ция    │                                                              │
│         │──────────────────────────────────────────────────────────────│
│         │  Footer (статус подключения, версия)                 28px   │
└─────────┴──────────────────────────────────────────────────────────────┘
```

---

## 1. Header

**Высота:** 40px, фиксированная

```
┌──────────────────────────────────────────────────────────────────────┐
│  AI Image Library                                      [☀] [⚙]     │
└──────────────────────────────────────────────────────────────────────┘
```

| Элемент  | Описание                                    |
| -------- | ------------------------------------------- |
| Название | «AI Image Library» — статичный текст (акцентный цвет) |
| [☀/🌙]  | Toggle светлая/тёмная тема (иконка меняется) |
| [⚙]     | Настройки (URL сервера, выбор пользователя) |

---

## 2. Sidebar (боковая панель)

**Ширина:** 60px свёрнутая / 200px развёрнутая

```
┌──────────────────────┐        ┌──────┐
│ [≡]                  │        │ [≡]  │
│                      │        │      │
│ 🖼  Галерея       ◄ │        │  🖼  │
│ ⚡  Генерация       │        │  ⚡  │
│                      │        │      │
└──────────────────────┘        └──────┘
  развёрнутая (200px)          свёрнутая (60px)
```

| Элемент          | Описание                                      |
| ---------------- | --------------------------------------------- |
| [≡]              | Toggle свёрнутая/развёрнутая                  |
| 🖼 Галерея       | Навигация на S-01                             |
| ⚡ Генерация     | Навигация на S-02                             |
| Активная полоска | 3px акцентная полоска слева у текущего пункта |

**Поведение:**

- Активный пункт: фоновая подсветка + акцентная полоска слева
- Hover: лёгкая подсветка фона
- В свёрнутом виде: только иконки, tooltip при наведении
- Состояние (свёрнутая/развёрнутая) сохраняется в настройках

---

## 3. Footer (статусная строка)

**Высота:** 28px, фиксированная

```
┌──────────────────────────────────────────────────────────┐
│  Сервер: ● Подключён                            v1.0.0  │
└──────────────────────────────────────────────────────────┘
```

В mock-режиме отображается «Сервер: ● Mock mode».

| Элемент        | Описание                                                  |
| -------------- | --------------------------------------------------------- |
| Статус сервера | ● Подключён (зелёный) / ● Отключён (красный) / ● Mock mode |
| Версия         | Версия приложения (правый край)                           |

---

## 4. Диалог настроек

Вызывается из header (⚙️):

```
┌─── Настройки ────────────────────────────────────┐
│                                                    │
│  URL сервера:   [http://localhost:8080        ]    │
│                                                    │
│  Пользователь:  [▼ Ivan (id=2)               ]    │
│                                                    │
│  [Проверить подключение]                           │
│                                                    │
│  Статус: ● Подключено                              │
│                                                    │
│  ─────────────────────────────────────────         │
│                                                    │
│  [ Отмена ]                   [ Сохранить ]        │
└────────────────────────────────────────────────────┘
```

Dropdown пользователей загружается из `GET /users`. Выбранный `UserId` передаётся серверу в HTTP-заголовке ко всем запросам.

## 5. Переиспользуемые компоненты

### Звёздочка избранного

```
☆ (не в избранном)  ←→  ★ (в избранном)
```

Клик → `POST/DELETE .../favorite` → toggle.

### Snackbar (уведомления)

```
┌─────────────────────────────────────────┐
│  ✅ Тип "Фотография" создан            │
└─────────────────────────────────────────┘
```

Появляется в нижней части экрана, исчезает через 3 секунды.
Типы: success (зелёный), error (красный), info (синий).

### Диалог подтверждения

```
┌─── Удалить тип? ──────────────────────────────┐
│                                                 │
│  Удалить тип "Фотография"?                     │
│  Все ассеты будут переназначены                 │
│  на "Неопределённый".                           │
│                                                 │
│  [ Отмена ]              [ Удалить ]            │
└─────────────────────────────────────────────────┘
```

### Placeholder (пустое состояние)

```
┌─────────────────────────────────────────┐
│                                          │
│        🖼                                │
│   Выберите тип изображения              │
│   для просмотра галереи                 │
│                                          │
└─────────────────────────────────────────┘
```
