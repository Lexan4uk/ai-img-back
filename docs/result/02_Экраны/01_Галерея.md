# S-01. Галерея (главный экран)

## Назначение

Просмотр сгенерированных изображений. Пользователь выбирает тип изображения,
опционально фильтрует по стилю — видит галерею ассетов от **всех** пользователей.

Это главный экран приложения — открывается при запуске.

## Блок 1: Выбор типа

**Расположение:** левая панель

| Элемент            | Описание                                                    |
| ------------------ | ----------------------------------------------------------- |
| Поиск              | Текстовое поле для фильтрации списка по имени               |
| Секция «Избранные» | Типы из `GET /image-types/favorites` — показываются первыми |
| Секция «Все»       | Остальные типы (без избранных), сортировка по ID            |
| ★ / ☆              | Звёздочка — toggle избранного (POST/DELETE `.../favorite`)  |

**Поведение:**

- При запуске: типы загружены, ни один не выбран. Блоки 2 и 3 скрыты / показывают placeholder.
- Клик по типу → загрузка галереи (`GET /assets?imageTypeId=...`) и показ блока 3 (стили).
- Активный тип подсвечен (фон).
- «Неопределённый» (id=1) показывается в списке, но визуально отмечен как системный (серый текст).

**Данные:**

- `GET /image-types` → список типов
- `GET /image-types/favorites` → ID избранных (для пометки ★)

## Блок 2: Галерея

**Расположение:** центральная область (основной контент)

| Элемент           | Описание                                   |
| ----------------- | ------------------------------------------ |
| Сетка изображений | Карточки с превью (до 7 в ряд, адаптивно)  |
| Карточка          | Превью изображения + название стиля + дата |
| Счётчик           | "Показано: N"                              |

**Карточка ассета:**

```
┌─────────────────┐
│                  │
│   [изображение]  │
│                  │
├──────────────────┤
│ Айвазовский      │
│ 28.02.2026       │
└──────────────────┘
```

**Поведение:**

- Загружается при выборе типа.
- При выборе стиля (блок 3) — перезагружается с `&styleId=...`.
- Сортировка: новые первые (по ID DESC, серверная).
- Клик по карточке → увеличенный просмотр (модальное окно или lightbox).

**Данные:**

- `GET /assets?imageTypeId=T` — все ассеты типа
- `GET /assets?imageTypeId=T&styleId=S` — с фильтром по стилю

**Маппинг карточки:**

`AssetDTO` содержит `styleId` (Long), но не название стиля. Клиент выполняет
локальный lookup по уже загруженному справочнику стилей (`GET /styles` — тот же
запрос, что используется для блока 3):

```
val stylesMap = styles.associateBy { it.id }   // Map<Long, StyleDTO>, один раз

// Для каждой карточки — O(1):
val styleName = stylesMap[asset.styleId]?.name
val date      = asset.createdAt
val image     = asset.fileUri
```

Справочник стилей — десятки записей, хранится в памяти. Дополнительных сетевых
запросов на карточку не требуется.

## Блок 3: Фильтр по стилю

**Расположение:** под блоком 1 (левая панель) или справа от блока 1

| Элемент       | Описание                                             |
| ------------- | ---------------------------------------------------- |
| Поиск         | Текстовое поле для фильтрации списка стилей по имени |
| «Все стили»   | Radio-кнопка — сброс фильтра (показать все стили)    |
| Список стилей | Radio-кнопки с именами стилей                        |
| ★ / ☆         | Toggle избранного стиля                              |

**Поведение:**

- Появляется после выбора типа в блоке 1.
- По умолчанию выбрано «Все стили» → галерея без фильтра.
- Поиск фильтрует список стилей по имени (case-insensitive).
- Выбор стиля → галерея перезагружается с фильтром.
- Избранные стили показываются первыми (без дублирования).

**Данные:**

- `GET /styles` → список стилей
- `GET /styles/favorites` → ID избранных

## Ошибки

| Ситуация                       | Отображение                                                                          |
| ------------------------------ | ------------------------------------------------------------------------------------ |
| Сервер недоступен              | Footer: ● Отключён (красный). Галерея показывает кешированные данные или placeholder |
| Нет ассетов для выбранной пары | «Нет изображений для этого сочетания»                                                |
| Ошибка загрузки                | Toast / snackbar: «Ошибка загрузки: ...»                                             |
